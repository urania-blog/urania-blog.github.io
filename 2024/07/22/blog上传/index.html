blog init

针对blog初始化会有一堆流程，为了省去不必要的麻烦，可以直接使用我发的压缩包

压缩包解压

下载依赖

项目根目录下，使用CMD执行npm install

替换账户

全局替换urania-blog为你的github账户名称

开发工具

建议使用typora

开启github blog的pages功能

进入github 仓库的settings项

如未开启pages，右侧会显示要你开启

配置私钥

CMD下执行ssh-keygen -t rsa -C "<yourEmailAddr>" -b 2048

一路默认即可

私钥和公钥信息会在C:/{你的windows账户名}/.ssh下生成

找到id_rsa.pub文件，记事本打开并复制里面的内容，下一步会用到

添加公钥信息

将上步复制信息，添加下图指引的地方【名称随意填】

博客上传

CMD下执行npm run kk，上传blog到你的github仓库下
